# Topic Modeling using Large Language Models

A research repository tracking the study of topic modeling techniques,
wge Language Mith a focus on Larodel (LLM)-based approaches.

## Research Scope

This repository documents a structured literature survey covering:

- Classical topic modeling methods (LDA, NMF, LSA)
- LLM-based topic modeling (BERTopic, PromptTopic, zero-shot approaches)
- State-of-the-art systems and applications
- Relevant code resources and libraries

## Repository Structure

| Folder    | Purpose                                |
| --------- | -------------------------------------- |
| `papers/` | Collected papers organized by category |
| `notes/`  | Reading notes per paper                |
| `report/` | Versioned research report              |
| `code/`   | Notebooks and code experiments         |
